# Gustavo Cardoso - Portfolio

Professional portfolio of Gustavo Cardoso, Control and Automation Engineer specializing in Industry 4.0, IIoT, and SCADA/MES Systems.

## Features
- **Modern UI**: Clean, mobile-first design using Vanilla HTML/CSS.
- **"PDF Magic"**: Built-in `@media print` rules ensure that pressing `Ctrl+P` generates a perfectly formatted, single-column resume. Web-only videos/GIFs are hidden, and high-quality static images are displayed instead, without page breaks slicing content.
- **Easy Customization**: Built with CSS variables for quick theming.

## Development

1. Clone the repository.
2. Open `index.html` in your browser. (Alternatively, use an extension like Live Server in VS Code).
3. The codebase is purely static and requires no build pipeline. 

## Deployment 
Ready to be deployed on GitHub Pages, Vercel, or Netlify.
